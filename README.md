# Hey-Yah-Projeto-das-Aulas-de-Ter-a-
🚀 Hey Ya! – Assistente de Produtividade Adaptativo
O Hey Ya! não é apenas mais um gerenciador de tarefas. É uma aplicação mobile pensada para quem vive a realidade de rotinas variáveis — plantonistas (12x36), freelancers e estudantes que precisam conciliar trabalho e desenvolvimento pessoal em horários não convencionais.

🎯 O Problema
A maioria das ferramentas de gestão de tempo é projetada para o padrão "comercial" (segunda a sexta, das 08h às 18h). Quando a rotina muda ou a escala de trabalho é alternada, o usuário perde o controle e o app se torna obsoleto.

✨ A Solução: Hey Ya!
Utilizamos inteligência artificial e uma arquitetura de dados flexível para criar um ecossistema que se ajusta à carga de energia e disponibilidade real do usuário.

Diferenciais:
Centralização Inteligente: Integração com calendários e listas de tarefas com priorização automática.

Modo 12x36 & Escalas: Ajuste automático da carga de estudos e lazer com base na escala de trabalho.

IA Orquestradora: Um assistente integrado via API da OpenAI que analisa o contexto para sugerir pausas, detectar sobrecarga e reorganizar a semana.

Gamificação: Sistema de progresso e recompensas para manter a consistência e o engajamento.

🛠️ Stack Tecnológica
Linguagem: Java (Android Nativo) – Garantindo performance e integração profunda com o sistema de notificações.

IDE: Android Studio.

Banco de Dados: MongoDB (NoSQL) – Armazenamento em documentos JSON para facilitar a comunicação fluida com a IA.

Inteligência Artificial: Integração com modelos GPT via API para processamento de linguagem natural e tomada de decisão.

🧠 Arquitetura e Dados
O uso do MongoDB é estratégico neste projeto. Ao trabalhar com documentos não estruturados, conseguimos enviar contextos complexos do usuário para a IA em formato JSON, permitindo que o assistente interprete padrões de comportamento e sugira mudanças na rotina de forma muito mais precisa que um banco relacional tradicional permitiria.

Nota de Prototipagem: Este projeto está sendo desenvolvido como parte da disciplina de Prototipagem, focando na criação de uma interface de usuário funcional, prazerosa e orientada a dados.
